# Expense Tracker Frontend

A React-based frontend for the Expense Tracker application that allows users to manage their personal expenses.

## Tech Stack

* React
* Axios
* CSS
* React Router

## Features

* User login and registration
* Add expense
* View expenses
* Delete expenses
* Filter expenses
* Pagination
* Dashboard summary

## Configuration

Create a `.env` file in the root directory and add the backend API URL.

Example:

REACT_APP_API_URL=http://localhost:8080/api

## Run Locally

Clone the repository

git clone <repo-url>

Install dependencies

npm install

Start the development server

npm start

Frontend runs on

http://localhost:5127
